### 요약
2D 미로 게임을 만들어서 최소 스패닝 트리를 활용해 컴퓨터가 미로 찾기를 하게 합니다.

### 개요
처음에는 사용자가 직접 미로를 빠져나가는 미로 찾기 게임을 만들려고 하였으나, 여러 알고리즘을 공부하다보니 최소 스패닝 트리를 사용해서 컴퓨터가 직접 미로의 길을 찾는 것도 
괜찮겠다 싶어 만들게 되었습니다.
